# stopBeingTurtle
Mediapipe기술을 활용한 거북목 진단 및 SNS 알림 서비스 프로그램

# 작업 사항

- 측면에서 목과 귀의 각도를 이용한 거북목 측정 프로그램
- 정면에서 얼굴 스케일 변화를 통한 거북목 측정 프로그램
- 정면에서 코와 명치를 기준으로 측정하는 거북목 측정 프로그램

## 참고 사항

- 측면에서 사용하는 프로그램은 정면 측정 프로그램의 신뢰도를 높이기 위한 용도임
